How to activate python environment : 

1. pip3 install virtualenv
2. python3 -m venv venv
3. source ./venv/bin/activate
4. pip install django
5. django-admin --version
6. django-admin startproject core .
The . means “create inside this directory” (so no nested folders).
7. python manage.py startapp main
(Create an app watchlist_app & added in setting in Installed_apps)
8. Create Models in model.py
9. Register model to admin panel

Note : 
# Step 1: You change models.py
python manage.py makemigrations

# Step 2: You apply changes to the actual DB
python manage.py migrate

python manage.py sqlmigrate watchlist_app 0001_initial
